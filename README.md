# JSONToolbox

A free, client-side JSON formatter, validator, tree viewer, minifier, diff tool, and YAML converter.

## Features

- **Format / Pretty-Print** — Indent JSON with 2 spaces, 4 spaces, or tabs
- **Minify** — Compact JSON to a single line with no whitespace
- **Real-Time Validation** — Instant syntax checking with descriptive error messages
- **Tree View** — Interactive collapsible tree with color-coded types (strings, numbers, booleans, null)
- **Diff Viewer** — Compare two JSON documents side by side with path-based change detection
- **JSON to YAML** — Convert JSON to YAML or YAML back to JSON
- **Copy & Export** — One-click copy to clipboard
- **Dark/Light Theme** — Persisted in localStorage

## Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+F` | Format JSON |
| `Ctrl+Shift+C` | Copy formatted JSON |
| `T` | Toggle theme |

## Tech Stack

- Vanilla HTML/CSS/JavaScript — zero dependencies, no build step
- Client-side only — no data sent to any server

## Run Locally

Open `index.html` in your browser. No install needed.
